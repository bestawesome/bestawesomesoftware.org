Open Source Baseboard Management Controller hardware and software

RunBMC

Dropbox released RunBMC, an open-source standard hardware and software interface for Baseboard Management Controllers (BMC). Dropbox donated the RunBMC specs, for the common hardware manufacturers, ASPEED and Nuvoton, to the Open Compute Project in August 2019.

The RunBMC design allows for swapping out BMCs separate from the rest of the board, isolating and locking down the BMC subsystem.

OpenBMC

OpenBMC has the widest range of support for various BMCs. (https://github.com/openbmc/openbmc). OpenBMC is a Linux distribution for embedded devices that have a BMC. OpenBMC is designed with a service-oriented approach and uses technologies such as Yocto, OpenEmbedded, systemd, and D-Bus.

U-Boot

The OpenBMC project encompasses u-boot, an open-source bootloader that boots a Linux kernel with a minimal root file system containing all the tools and binaries needed to run OpenBMC.

https://www.denx.de/wiki/U-Boot

https://en.wikipedia.org/wiki/Das_U-Boot

u-bmc

u-bmc uses u-root to create a Linux OS distribution that is fully open-source. u-bmc (https://github.com/u-root/u-bmc) has a more minimal BMC software architecture by replacing IPMI with gRPC. u-bmc borrows and contributes to OpenBMC. Unlike OpenBMC, u-bmc boots a Linux kernel directly from the ASPEED startup code after DRAM initialization to remove the need for a bootloader such as u-boot. u-bmc is still in experimental stage.

https://u-bmc.readthedocs.io/en/latest/
